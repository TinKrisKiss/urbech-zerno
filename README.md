🔗 **Живой сайт:** https://tinkriskiss.github.io/urbech-zerno/

---

## 📌 О проекте

**ЗЕРНО** — это сайт-витрина для бренда натурального урбеча и ореховых паст холодного отжима. Проект демонстрирует, как с помощью современных веб-технологий можно создать полноценный интернет-магазин с корзиной, каталогом товаров и оформлением заказа без использования бэкенда.

**Целевая аудитория:** ЗОЖ-активисты, ценители натуральных продуктов, молодые мамы.

**Стиль:** Эко/Крафт с премиальными акцентами.

## 🛠 Технологии

### Frontend
- **HTML5** — семантическая разметка
- **Tailwind CSS** — утилитарный CSS-фреймворк
- **Alpine.js** — реактивность и интерактив

### Инструменты
- **VS Code** — редактор кода
- **GitHub Pages** — хостинг и деплой
- **Git** — контроль версий

### AI-инструменты
- **Midjourney / DALL-E** — генерация изображений продуктов
- **RealFaviconGenerator** — создание favicon

## ✨ Функционал

### Главная страница
- ✅ Hero-секция с адаптивным фоном (отдельное изображение для мобильных)
- ✅ Блок «Проблема/Решение» с контрастными плашками

- ✅ Секция «Состав/Процесс» с фоновым изображением

- ✅ FAQ с аккордеоном и фоновым изображением

### Страница товара
- ✅ Динамическая загрузка данных через URL-параметр (`?id=`)

- ✅ Блок КБЖУ с визуальными карточками

- ✅ Изменение количества, удаление товаров
